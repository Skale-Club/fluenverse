# Fluenverse - Base Next.js

Estrutura genérica para iniciar um projeto com Next.js (App Router) e TypeScript.

## Rodando localmente

```bash
npm install
npm run dev
```

Aplicação em `http://localhost:3000`.

## Estrutura

```text
.
|-- app/
|   |-- about/page.tsx
|   |-- api/health/route.ts
|   |-- globals.css
|   |-- layout.tsx
|   `-- page.tsx
|-- components/
|   `-- section.tsx
|-- lib/
|   `-- utils.ts
|-- public/
|   `-- logo.svg
|-- .eslintrc.json
|-- .gitignore
|-- next.config.mjs
|-- package.json
|-- tsconfig.json
`-- README.md
```
